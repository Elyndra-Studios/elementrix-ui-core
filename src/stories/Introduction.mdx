## Component Categories

### Atoms (7)
Basic building blocks: buttons, inputs, labels, badges, checkboxes, switches, textareas

### Molecules (19)
Composite components: cards, alerts, tooltips, menus, tabs, accordions, progress bars, chips

### Organisms (20)
Complex components: modals, drawers, tables, carousels, forms, calendars, pickers, grids

### Layout (5)
Layout primitives: containers, grids, stacks, spacers, aspect ratios

### Utilities (2)
Helper components: portals, transitions

## Design System

Built on a comprehensive design token system with:

- **Colors**: Neutral palette + primary brand + semantic colors
- **Typography**: Sans & mono font families with 9 size scales
- **Spacing**: 8px base grid with 13 spacing values
- **Shadows**: 7 elevation levels
- **Border Radius**: 8 radius options
- **Transitions**: Smooth animations with 3 duration presets

## Theme Support

Toggle between light and dark themes using the toolbar above.

```html
<body data-theme="dark">
  <el-button>Dark themed button</el-button>
</body>
```
